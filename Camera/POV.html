Point de vue

Comment ça marche ?

Au cinéma, le point du vue est une technique narrative permettant de filmer le point de vue d'un personnage. Il existe différents types de points du vue:

  • Le point de vue omniscient: dans ce cas, le narrateur en sait plus que les autres personnages.
  • Le point de vue interne: dans ce cas, on ne sait que ce que dit ou fait le personnage.
  • Le point de vue externe: dans ce cas, le personnage en sait plus que le narrateur.

    • Aller plus loin

    Le réalisateur va utiliser l’un de ces mouvements pour révéler au spectateur le point de vue d’un personnage ou d’un objet. Ce que le spectateur verra à l’écran représente exactement ce que le personnage voit. Cette technique est idéale pour créer l’identification au personnage. Elle peut au contraire aussi être utilisée pour créer une angoisse chez le spectateur, une ambiance menaçante.
